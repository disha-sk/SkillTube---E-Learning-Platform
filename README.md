# SkillTube

![PHP](https://img.shields.io/badge/PHP-7.4-blue?logo=php) ![MySQL](https://img.shields.io/badge/MySQL-8.0-orange?logo=mysql) ![Live Demo](https://img.shields.io/badge/Live-Demo-green)

SkillTube is a web application designed for teachers and students. Teachers can upload academic and skill videos, resources, and articles, and conduct group discussions. Students can browse and watch videos, access resources, upload and view projects, and participate in discussions. The platform is built with PHP, MySQL, and Laragon, featuring a responsive and user-friendly interface.

---

## Features

### Teacher
- Upload and manage academic videos
- Upload and manage skill videos
- Upload resources and articles
- Conduct and manage group discussions

### Student
- Browse and watch uploaded videos
- Access resources and articles
- Upload and view projects
- Participate in group discussions

---

## Tech Stack
- **Backend:** PHP  
- **Database:** MySQL  
- **Local Development:** Laragon  
- **Frontend:** HTML, CSS, JavaScript
